# Ideas

> 이 문서는 구현 여부가 아직 정해지지 않은 아이디어를 자유롭게 기록한다.

## 기능 아이디어

- 출퇴근 시간 기준 추천
- 직장 위치 기준 추천
- 학군 기반 비교
- GTX와 신규 교통망 정보
- 지역 간 격차 비교
- 여러 목표 지역 동시 비교

## 분석 아이디어

- 급지 상승·하락 기록
- 지역별 장기 성장률
- 지역 간 격차의 평균 회귀
- 현재 격차의 역사적 백분위
- 가격 상승기와 하락기 비교
- 거래량 변화
- 신축과 구축 분리 분석
- 전용면적별 가격 비교

## 보류

### 개인 자금과 대출 기반 계산

- 보유 자금, 대출 가능 금액, 필요한 추가 자금은 계산하지 않는다.
- 개인별 대출 조건 차이가 커서 객관적인 결과를 제공하기 어렵다.
- 서비스는 급지별 평균 평단가와 지역 간 격차에 집중한다.

### 저평가 지역

### 저평가 지역

- '저평가'라고 단정하지 않는다.
- 비슷한 급지나 조건의 지역과 가격 차이만 보여준다.
- 최종 판단은 사용자가 하도록 한다.
